# [Tide 🪼](https://gitee.com/oschina/tide)

开箱即用、扩展性强、支持 Markdown 语法、基础功能完善的 React 富文本编辑器。

## 在线演示

https://oschina.gitee.io/tide

![](./docs/images/tide-screenshot.jpg)

## 特性

- 支持 Markdown 快捷语法
  - 标题
  - 引用
  - 代码块
  - 表格
  - 超链接
  - 图片
  - 分割线
  - 粗体、斜体、删除线
  - 无序列表、有序列表、任务列表
- 支持粘贴 Markdown 文本
- 支持功能增强的 Table 表格
- 支持粘贴、拖拽图片
- 支持 Emoji 表情选择
- 内建菜单栏
- 支持多主题
